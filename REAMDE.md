
go-zero cursor 

## 配置方法

```Bash
# 添加 ai-context 作为 submodule
git submodule add https://github.com/zeromicro/ai-context.git .cursorrules

# 更新到最新版本
git submodule update --remote .cursorrules
```

Cursor 会自动读取 .cursorrules 目录下的所有 .md 文件作为项目规则。
